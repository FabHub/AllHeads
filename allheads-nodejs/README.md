# How to run Node.js server for frontend?

## Prerequisities

1. Node version atleast v20.11.0
2. NPM version atleast v10.2.3

## Steps to run
1. got to the allheads-backend folder
2. Create .env file in the root of allheads-backend with content `AZURE_CONNECTION_STRING={blob_storage_connection_string}`
3. run `npm install`
4. run `npm start`
5. if everything goes right, let it running in one terminal and go start the allheads react app
